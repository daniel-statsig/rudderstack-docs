# Statsig

Statsig helps companies safely A/B test features in production before rolling them out, avoiding product debates and costly mistakes when shipping out new features. Statsig automates the grunt work so that A/B tests are always running automatically and you always know how your features are performing.

<div class="successBlock">

**Find out more in the <a href="https://docs.statsig.com/integrations/data-connectors">Statsig Documentation</a>.**

</div>

## Getting Started

1. From the Destinations catalog page in the Rudderstack App, click "New destination".
1. Search for “Statsig” in the Destinations Catalog, and select the “Statsig” destination.
1. Name your new Destination.
1. Choose which Source(s) should send data to the “Statsig” destination.
1. Enter the Statsig “Server Secret Key” in the "Server Secret Key" field.
   - This “Server Secret Key” can be found in the <a href="https://console.statsig.com/">Statsig Console</a>, under Settings > API Keys.
